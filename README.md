<i>Catharsis in apocalypse is a 2-D massive multi-palyer RPG (MMORPG) game built with Godot engine.</i>


## Features

- Character selection
- Occupies less space
- Cool outfits
- Amazing weapons
- Talk to players (chat system)
- Marriage system
- Dragons
- unicorns
- Underwater gaming
